About AMS

Overview

The Attendance Management System (AMS) is a modern, responsive web application designed to streamline the process of taking and managing student attendance using dynamic QR codes.

Key Features

  • Role-based access control (Admin, Faculty, Student)
  • Dynamic, time-limited QR code generation for secure attendance
  • Real-time attendance tracking and dashboard analytics
  • Secure password hashing and recovery workflows
  • Responsive glassmorphism UI design

Technology Stack

Python (Flask) MySQL HTML5 / Vanilla CSS Werkzeug Security
